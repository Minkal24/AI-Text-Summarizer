# Transformer-Based AI Text Summarizer

This project summarizes long text using a fine-tuned T5 Transformer model.

## Technologies

- Python
- FastAPI
- Hugging Face Transformers
- PyTorch
- HTML
- CSS
- JavaScript

## Features

- AI-powered text summarization
- FastAPI backend
- Responsive user interface
- Transformer-based NLP model
